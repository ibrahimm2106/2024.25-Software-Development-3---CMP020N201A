## Installation
1. Ensure you have the following installed on your system:
   - Java Development Kit (JDK) 11 or higher
   - Maven or Gradle (depending on the project's build system)
2. Navigate to the project directory:
   ```bash
   cd 2024.25-Software-Development-3---CMP020N201A
   ```
3. Build the project using the appropriate build tool:
   - For Maven:
     ```bash
     mvn clean install
     ```
   - For Gradle:
     ```bash
     gradle build
     ```

## Usage
After building the project, you can run the application using:
```bash
java -jar target/<your-jar-file>.jar
```
Replace `<your-jar-file>` with the name of the generated JAR file.
